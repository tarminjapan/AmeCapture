# ルール

- 文章は**日本語**で記述する。
- テキストにGemini Code Assistを呼び出すコマンドを記述する。
  - `/gemini review`
  - `/gemini summary`
  - `/gemini`
  - `/gemini help`

## 【AIエージェントの絶対ルール】

- コードを追加・変更した後は、必ず `npm run check:all` を実行すること。
- Lintエラー、型エラー、Clippyのエラーが1つでも発生した場合、**いかなる理由があっても `#[allow(...)]` や `#![allow(...)]` を使ってエラーを握り潰してはならない。** 必ずコードの根本原因を修正すること。
- エラーが完全に0になるまで、ユーザーに作業完了の報告を行わずにAI自身で修正のループを回すこと。
