# 4. MVPの画面一覧・画面遷移図

## 4.1 MVP画面一覧

### 画面1: メインウィンドウ（ワークスペース）

**役割:** アプリの母艦\
**主機能:**

- アイテム一覧表示
- サムネイル表示
- ソート
- 削除
- 名前変更
- お気に入り
- ダブルクリックで編集画面へ
- 右クリックメニュー（コピー、保存先を開く、削除）

#### UIイメージ

```text
+--------------------------------------------------------------+
| [新規キャプチャ] [検索] [フィルタ] [設定]                     |
+--------------------------------------------------------------+
| サムネイル一覧                                               |
| [img1] [img2] [img3] [img4] ...                             |
|                                                              |
|                                                              |
+--------------------------------------------------------------+
| 詳細ペイン                                                   |
| タイトル / 日時 / 種別 / 保存先 / タグ / 操作ボタン          |
+--------------------------------------------------------------+
```

### 画面2: 領域選択オーバーレイ

**役割:** 範囲キャプチャ時に表示\
**主機能:**

- 画面全体半透明表示
- ドラッグで範囲選択
- 選択サイズ表示
- Escでキャンセル

#### UIイメージ

```text
+--------------------------------------------------------------+
| 半透明オーバーレイ                                           |
|                                                              |
|      ┌──────────────────────────────┐                       |
|      │ 選択範囲                      │                       |
|      └──────────────────────────────┘                       |
|                                                              |
|  Esc: キャンセル                                             |
+--------------------------------------------------------------+
```

### 画面3: 編集画面

**役割:** 注釈編集\
**主機能:**

- 矢印
- テキスト
- 四角形
- モザイク
- トリミング
- Undo / Redo
- 保存
- コピー

#### UIイメージ

```text
+--------------------------------------------------------------+
| [保存] [コピー] [Undo] [Redo] [矢印] [テキスト] [四角] ...   |
+--------------------------------------------------------------+
|                                                              |
|                    画像キャンバス                            |
|                                                              |
|                                                              |
+--------------------------------------------------------------+
| プロパティ（色/太さ/文字サイズ等）                           |
+--------------------------------------------------------------+
```

### 画面4: 設定画面

**役割:** 基本設定\
**主機能:**

- 保存先設定
- 画像形式設定
- ホットキー設定（将来）
- 起動時常駐
- ファイル命名規則（将来）

### 画面5: タスクトレイメニュー

**役割:** 常駐中の起点\
**主機能:**

- 範囲キャプチャ
- 全画面キャプチャ
- ウィンドウキャプチャ
- ワークスペースを開く
- 設定
- 終了

## 4.4 ユースケース別の導線

### 導線A: 普通にキャプチャして貼り付けたい

1. ホットキー押下
2. 範囲選択
3. 保存
4. 自動でワークスペース登録
5. クリップボードコピー
6. Teams / Officeへ貼り付け

### 導線B: バグ報告用に注釈を入れたい

1. キャプチャ
2. ワークスペース一覧から対象選択
3. 編集画面を開く
4. 矢印・テキスト・モザイク追加
5. 保存
6. コピー or ファイル添付

### 導線C: 後で見返したい

1. キャプチャ
2. ワークスペース自動登録
3. 名前変更
4. お気に入り
5. 必要時に検索・再利用
