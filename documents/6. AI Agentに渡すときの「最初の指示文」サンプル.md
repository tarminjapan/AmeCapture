# 6. AI Agentに渡すときの「最初の指示文」サンプル

以下、そのまま使えるように書いておきます。

```text
Windows向けのローカル完結型スクリーンキャプチャアプリをRust + Tauriで実装したいです。
Screenpressoライクですが、クラウド連携は実装不要です。
むしろローカル保存とワークスペース機能を重視します。

必須条件:
- Windows向け
- Rust（バックエンド）
- Tauri v2採用
- React + TypeScript（フロントエンド）
- Tauri Commands + State管理構成
- SQLite使用（rusqlite / sqlx）
- ローカルファイル + SQLiteメタデータ管理
- クラウド機能なし
- 最初からワークスペース機能あり

MVP機能:
- 指定領域キャプチャ
- 全画面キャプチャ
- ウィンドウキャプチャ
- ワークスペース一覧
- サムネイル表示
- 削除 / 名前変更
- クリップボードコピー
- タスクトレイ常駐
- グローバルホットキー
- 編集機能（矢印、テキスト、四角形、モザイク、トリミング）
- 保存
- Undo/Redo
```
